Quartz classes act a bit different than traditional classes.

In Quartz all fields are public read accessable, reducing the need to implement
a ton of get functions that most the time just return the variable anyways.

If you want fields to be hidden, you mark the varibale with the qualifier
'private'

In contrast functions are private by default. To make the public, just mark them
with the 'public' qualifier.

```
public class Example {

    uint id;
    String name;
    String message;

    private static uint next_id = 0;

    public Example(String name, String message){
        this.name = name;
        this.message = message;
        next_id += 1;
        this.id = next_id;
    }

    public String toString() {
        return String.format(
            "Example:{id: %0, name: %1, message: %2}",
            [this.id, this.name, this.message]
        );
    }

}
```
